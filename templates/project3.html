003 : Freddie Mac

Freddie Mac is one of largest purchasers in the Secondary Mortgage Market. They use capital to expand the secondary market for mortgages in the US.

My role at Freddie Mac is Automation Software Engineer.

Role:

My role was to create automation script for specific jobs within the teams and company.

I was to create scripts to run routine tasks like ensuring data accuracy. If there was a discrepancy between data, the scripts would notice it and execute additional scripts to update the data correctly. I would keep track of record count, accuracy, and completeness.

Languages:

I used Java as my scripting language, PowerShell to help execute commands on remote services, and I used SQL for ensuring Data accuracy.

Complete list of Duties:

  • Create Automation Testing Scripts for specific jobs
  • Ensure Data accuracy
  • Demo new scripts that could be widely used
  • Find and Resolve any discrepancy in company data

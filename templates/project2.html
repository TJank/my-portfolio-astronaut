002 : Wells Fargo

Wells Fargo is a huge banking corporation known worldwide. They offer standard banking services, mortgage lending, credit cards, business loans, and investments.

My role at Wells Fargo is Software Automation Engineer.

Role:

Initially I was a Software Automation Engineer to create automation test script to test a variety of company applications. On my first project I was quickly moved to the Lead Automation Engineer and was the direct point of contact for my team and Wells Fargo Managers.

I worked on several projects during my time with Wells Fargo and learned how to start and automation project from nothing or to be able to quickly join in on how a current team was handling things.

Languages & Tools:

My team and I used Java with Selenium as our scripting language to automate test cases. We deployed those test scripts using a Jenkins server to execute tests at certain dates and times.

We also used tools like ALM (Application Lifecycle Management) by Microsoft to trace our tests as well as the pass/fail results when executed. ALM integration allowed managers of all levels to quickly view our results and progress.

Complete list of Duties:

  • Create Automation Testing Project and Scripts
  • Convert 800 Regression, 120 Unit, and 50 Smoke manual tests into Automation Scripts
  • Execute Automation Scripts and report results to managers and developers
  • Configure and manage Jenkins server to execute scripts

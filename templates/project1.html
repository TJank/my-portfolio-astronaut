001 : Eastdil Secured

Eastdil Secured is a real estate investment bank that does business all over the world. They combine commercial real estate and capital markets expertise.

My role at Eastdil is Senior QA Automation Engineer.

Role:

My role was to create automation script for company software and applications, as well perform quality assurance during code changes/updates.

I was a part of the Deployment Stages and had to validate that no application breaking bugs or undesired result was created due to recent updates. I completed these tasks by using Regression Automation Scripts as well as manual testing when needed.

Languages:

We used Python as our main scripting language and Playwright as our automation library. Playwright is a relatively new automation tool that can be coded in Python, Java, .NET, and JavaScript

Complete list of Duties:

  • Create Automation Testing Scripts
  • Run Regression Tests during Deployment stages
  • Validate Code Changes to be Expected Result
  • Find and Report Bugs introduced in changes
  • Validate the Look, Flow, and Functionality of Application and Software
